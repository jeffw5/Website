owl:Class Taxonomy

Concepts

✏️ Rename Taxonomy URI

⚠️ Migrates this taxonomy's owl:Class members (via rdfs:isDefinedBy) to the new URI and moves the named graph. Updates all triples where this URI appears as subject, object, or named graph.

⊕ Set Named Graph

Typically the same as the Asset URI.
⚠️ Moves matching triples from the default graph into the specified named graph.
